# Campus Placement Management System

## Project Overview

The Campus Placement Management System is a web application developed using Java, JDBC, Servlets, JSP, and MySQL. The system automates campus recruitment activities and manages students, companies, placement drives, and applications.

## Objectives

* Automate placement activities.
* Reduce manual work.
* Maintain student records.
* Manage company recruitment drives.
* Generate placement reports.

## Actors

### Admin

* Manage students
* Manage companies
* Manage placement drives
* Generate reports

### Student

* Register and login
* Update profile
* Apply for placement drives
* View application status

## Modules

### Authentication Module

* Registration
* Login
* Logout
* Session Management

### Student Management Module

* Add Student
* Update Student
* Delete Student
* Search Student

### Company Management Module

* Add Company
* Update Company
* Delete Company

### Placement Drive Module

* Create Drive
* Update Drive
* Register Students

### Reports Module

* Placement Reports
* Company Reports
* Drive Reports
